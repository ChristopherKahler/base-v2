    - `get(cwd, ns, slug)` → SELECT all properties
    - `update(cwd, ns, slug, status, description)` → field_update pattern
    - IRI: `{ns.uri}entity/{slug}`

    **1c. Update crud/mod.rs:** Add `pub mod decision; pub mod entity;`

<task type="auto">
  <name>Task 2: Goal and Reminder CRUD</name>
  <files>src/crud/goal.rs, src/crud/reminder.rs, src/crud/mod.rs</files>
  <action>
    **2a. Goal module (`src/crud/goal.rs`):**
    - `add(cwd, ns, name, target)` → INSERT with Goal type, name, description=target, createdAt
    - `list(cwd, ns)` → SELECT all Goals
    - `update(cwd, ns, slug, status, target)` → field_update pattern
    - IRI: `{ns.uri}goal/{slug}`

    **2b. Reminder module (`src/crud/reminder.rs`):**
    - `add(cwd, ns, name, due_date)` → INSERT with Reminder type, name, dueDate (xsd:date), createdAt
    - `list(cwd, ns)` → SELECT all Reminders ordered by dueDate
    - `remove(cwd, ns, slug)` → DELETE all triples for the reminder IRI (actual deletion, not soft-delete)
    - IRI: `{ns.uri}reminder/{slug}`

    For remove, SPARQL: `DELETE WHERE { GRAPH ?g { <iri> ?p ?o } }` — removes all triples about the entity.

    **2c. Update crud/mod.rs:** Add `pub mod goal; pub mod reminder;`
  </action>
  <verify>`cargo check` passes. `cargo test` existing tests green.</verify>
  <done>AC-3 and AC-4 satisfied at module level.</done>
